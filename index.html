Import Data
📥 Import from Garmin Connect - supports Activities CSV, sleep JSON, and full data exports.
Accepts: activities JSON, summarizedActivities.json, wellness JSON
How to get your Garmin data:

📊 Activities (CSV): connect.garmin.com → Activities → click ↓ export button → "Export to CSV" → upload here

😴 Sleep + Full export (JSON): connect.garmin.com → profile icon → Account Settings → Data Export → Request Data → download ZIP → extract → upload summarizedActivitiesExport.json, sleepData.json, and dailySleepDTO.json

Accepts: .csv and .json files
